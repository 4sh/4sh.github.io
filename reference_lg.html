LG France DMS Mobility, application mobile pour la gestion des interventions SAV.
 

Le SAV de LG France souhaite doter ces techniciens d’une application mobile leur permettant d’interagir facilement et rapidement avec le SI LG.

Le client confie à 4SH la conception et la réalisation de cette application mobile.

 
 
  • Framework .NET 3.5 et Framework Compact .NET 3.5
  • MS SQL Server et MS SQL Compact Edition
  • MuleESB
 

Ce projet permet au technicien de procéder de manière beaucoup plus réactive grâce à :

  • Un planning des interventions actualisé,
  • Un historique des réparations lui permettant de prévoir le matériel nécessaire,
  • Une automatisation des processus SAV de LG,
  • Une gestion des pièces, facilitée avec le scanner de code barre du PDA.
