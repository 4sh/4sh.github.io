SUPERPLASTIC Un site e-commerce de vente d'accessoires en plastique à destination des professionnels et des particuliers.
 

Superplastic a souhaité refaire son site e-commerce afin de le moderniser et de le synchroniser avec son ERP (Mercator). Le rôle de 4SH a été de prendre en compte ses contraintes, de proposer une solution efficace pour la gestion du catalogue et respecter le design du nouveau site, défini par Superplastic dans un wireframe.

 
 

La partie serveur est développée avec des technologies Java telles que Freemarker, Restlet, Guice et Guava. ElasticSearch est utilisé, entre autre, pour gérer le catalogue synchronisé et ainsi permettre une recherche efficace des produits. Pour la partie client, nous avons utilisé les technologies de base : HTML, CSS et Javascript. En revanche, le site a été conçu en "Responsive design" à l'aide de CSS 3 afin de proposer une interface plus adaptée sur certains écrans, tels les mobiles.

 

4SH a su relever les multiples défis de ce projet, à savoir :

  • créer le site à partir du wireframe tout en permettant du responsive design
  • gérer la synchronisation avec l'ERP Mercator.
  • utiliser ElasticSearch
